Amazing Features

Total Inventory Control

Manage a Shopping List, Pantry, and Fridge from the same app using a native Tab Bar. Swipe right on any item to instantly move it between lists, or use bulk actions to move entire lists at once. Smart merging intelligently combines quantities for existing items, while date stamps track exactly when each item was added to your inventory.

AI-Generated 3D Icons

Experience custom, high-quality 3D icons for every item, powered by on-device Apple Intelligence.

Predictive Text Suggestions

Leverage on-device LLMs for intelligent grocery suggestions as you type, making list creation faster.

Gesture-Driven UX

Enjoy a modern, intuitive interface with swipe gestures, pull-to-add, and long-press menus for effortless control.
